Ett centralt attribut vid hantering av layouts är position, med vilket webbläsaren får veta hur ett element ska hanteras på sidan. Tillgängliga värden är static, absolute, relative samt fixed.

static
Detta är standardvärdet för position. Positionen är ej specificerad och alltid anpassad efter sidans "normala" flöde
absolute
Med detta värde sätts elementets position i förhållande till närmast omgivande element. Exempelvis kan en bild anges att alltid vara placerad i övre högre hörnet av en <div>
relative
I motsats till static anpassar sig element nu inte efter sidans normala flöde utan relativt till sin ursprungliga placering, till exempel med 15 pixlars marginal från vänsterkanten
fixed
Elementet förhåller sig till webbläsarens "viewpoint" (det område som avser den visade webbsidan). Om sidan scrollas visas fortfarande elementet på samma plats i viewpointen
float
Med attributet float placeras ett element till höger eller vänster i det omgivande elementet, genom att ange left eller right. Anges flera element med exempelvis float: left kommer de lägga sig bredvid varandra i den mån de får plats. Vid sidan av flytande element kan andra element fortsätta flöda. Är detta icke önskvärt kan attributet clear användas med möjliga värdena left, right samt both. Med vänster och höger tillåts inga andra flytande element på den sida som anges. Vill ett elemt hamna där i webbläsarens normala flöde kommer radbrytning att ske. Anges both tillåts inga andra element på någon sida och radbrytning kommer alltid att ske
clear
Se förklaring under float
z-index
Med z-index anges visningshierarkin för element OM de skulle ta upp samma visuella utrymme, genom att ange en siffra. Ju högre siffra desto högre prioritering. Ett element med z-index: 2 kommer till exempel hamna över ett element med z-index: 1 om de överlappar varandra. Detta kan jämföras med en sorterad kortlek där det högsta värdet ligger överst och därför är det som syns

Storlek på element är något som behöver göras. För att hantera detta kan nedan attribut användas:

width
Elementets bredd, som anges i tidigare presenterade enheter
height
Som width fast det är elementets höjd som anges
min-width / min-height
Den minsta bredd/höjd som elementet kan få. Växer innehållet så att det inte får plats kommer elementet växa
max-width / max-height
Den maximala bredd/höjd elementet kan få. Växer innehållet så att det inte får plats när max-värdena har uppnåtts kommer elementet inte fortsätta att växa
vertical-align
Detta attribut används för att specificera hur ett element ska placeras på vertikalen. Detta exempel är ett span-element där vertical-align: text-top har angivits. TEST För att visa elementet tydligare har också kantlinjen färgats samt tilldelats större fontstorlek. Som påvisas placeras elementet med toppen i höjd med omgivande textens överkant
Fixerad layout

Med fixerad layout anges fasta värden för elementens storlek. Minskas eller ökas fönstrets storlek ändras inte elementens storlek. Exempel på fixerad layout. För att uppnå detta används en absolut enhet, till exempel px eller mm.

Flytande layout

Med flytande layout har elementen storlek angivet i procent i förhållande till omgivande element. Minskas fönstret så minskas elementen, ökas fönstrets storlek så sker samma sak med elementen. Exempel på flytande layout

Elastisk layout

Elastisk layout kan användas för sidor där användaren av någon anledning behöver större textstorlek för att kunna läsa sidan. Text flödande i kolumner

I CSS finns också ett layoutalternativ i form av kolumnering. Antalet kolumner anges med attributet column-count och varje kolumns bredd anges med attributet column-width. I exemplet nedan är all text skrivet i ett stycke, som tilldelats 10 rader som tar upp 10% av sidans bredd. Höjden på kolumnerna avgörs av hur mkt text som får plats i respektive kolumn, vilket fördelas jämt mellan dem.
