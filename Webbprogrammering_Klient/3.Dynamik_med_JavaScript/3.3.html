Solna AIK

toggle

Genom funktionen toggle() kan alternering mellan att gömma och visa ett element göras. I exemplet nedan försvinner cirkeln när knappen klickas på för att vid nästa klick visas igen.

hover

Med funktionen hover() går det direkt att alternera mellan mouseover och mouseout. I exemplet nedan får cirkeln en färg när markören är över den och en annan när markören är utanför cirkeln

Tangentbordshändelser

I exemplet nedan kan du prova att flytta cirkeln både i sidled och vertikalt genom att använda piltangenterna. Keydown används med keyCode för att identifiera att rätt tangent används vid önskad förflyttning. Dessutom exemplifierar utvinning av information genom att elementets ursprungliga position hämtar från css, genom margin-left samt margin-top. Det är i förhållande till detta hämtade värde som förflyttningen sedan sker genom omräkning av marginalerna.
