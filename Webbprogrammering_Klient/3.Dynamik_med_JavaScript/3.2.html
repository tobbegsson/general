Uppgift 3.2 - JavaScript-objekt

Array

Här kan du lägga till variabler i en lista. Lägg till hur många du vill. Sedan kan du kontrollera vilka variabler du har lagt till

Boolean

Skriv in ett tal och klicka på knappen för att se om talet är större eller mindre än 10. Om talet är större visas "true", om mindre visas "false"

Date

Klicka på knappen nedan för att visa dagens datum

Math

Klicka på knappen nedan för att slumpa fram ett värde mellan 0 och 10.

Number

Skriv in ett heltal och klicka på knappen för att få veta dess värde även som hexadecimalt samt binärt tal

String

Skriv in en text eller ett och klicka på knappen nedan för att få fram antal tecken

RegExp

Texten nedan är från en av tidernas vackraste låtar. Skriv in en bokstav eller ett ord och klicka på knappen för att få fram hur många gånger det förekommer i texten

Global
