Metoder

alert, conform, prompt

Klicka på knapparna nedan och se vad som händer




open, close, focus, blur

Klicka på knapparna för att öppna och stänga en intressant sida, samt för att tilldela eller ta bort focus från fönstret


Metodenfocus är default när ett nytt fönster öppnas, motsatsen blur kan exempelvis användas om man vill att fönstret ska öppnas i bakgrunden

set-/clear -Interval/-Timeout

Detta är en klocka som uppdateras varje sekund med hjälp av metoden setInterval:

...Och med denna knappen kan klockan stannas, genom metoden clearInterval

Här återanvänds metoden alert() tillsammans med setTimeout vilken är satt till 5000 milisekunder (5 sekunder) från att du tryckt på knappen. Innan dessa 5 sekunder har passerat kan du trycka på stoppknappen för att avbryta att alert-fönstret kommer upp

moveBy, moveTo

Klicka upp fönstret och styr det sedan åt det håll som önskas. Förflyttningen sker genom metoden moveBy(). Flytten till fast position sker genom moveTo()





resizeBy, resizeTo

Öppna fönstret med hjälp av knappen. Du kan sedan välja att förminska fönstret med 10x10 pixlar eller förstora det med 15x15. Dessutom kan du tilldela den fasta storleken 250x250

scroll, scrollBy, scrollTo

OBS! Metoden scroll har numera ersatts av scrollTo. Denna metod exemplifieras i knappen '100 pixlar från toppen'. Knapparna 'Uppåt' och 'Nedåt' är exempel på scrollBy där varje knapptryck genererar 5 pixlars scrollning



print

Klicka på knappen för att skriva ut denna sida
